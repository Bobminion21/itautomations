Frequently Asked Questions

How long does a typical project take?
Most Google Sheets automations are delivered in 3-7 days. Larger projects like bot systems or multi-platform integrations typically take 1-2 weeks.
Do I need technical knowledge to use your products?
No. All products include setup instructions written for non-technical users. Templates install with a one-click setup function. Consulting projects come with full documentation.
What if I need changes after delivery?
Minor adjustments are included with every project. For significant scope changes, we provide a transparent quote for the additional work before proceeding.
Do you offer ongoing maintenance?
Our digital products are designed to run independently with zero maintenance. For custom consulting projects, ongoing support arrangements can be discussed.
What payment methods do you accept?
Digital products are sold through our store (credit card, PayPal). Consulting services can be invoiced through Upwork, Fiverr, or direct invoice.
How do the Google Sheets templates work?
Each template comes with Apps Script files (Code.gs and SheetSetup.gs). Simply copy them into a new Google Sheets project via the Apps Script editor, run the one-click setup function, and your fully formatted, automated spreadsheet is ready to use.
Do I need coding experience?
Not at all. Every template includes a one-click setup function that creates all sheets, formatting, and automation for you automatically. Just copy, paste, and click setup. Detailed README instructions are included with every download.
Can I customize the templates?
Yes, absolutely. Full source code is included with every product. You can modify the scripts, adjust formulas, change formatting, add new features, or adapt the templates to your specific workflow. The code is clean and well-commented.
What's included in each download?
Each product download includes: Code.gs (the main automation script), SheetSetup.gs (the one-click setup function that builds your spreadsheet), and a detailed README with step-by-step installation and usage instructions.
Do you offer refunds?
Yes. We offer a 30-day money-back guarantee on all digital products. If you're not satisfied for any reason, contact us at peter@thewyattclan.com and we'll process your refund promptly, no questions asked.
